# whichttp

A CLI for looking up http status codes by code

## Installation

1. Clone this repository
2. That's it. No dependencies other than Crystal.

## Usage

Build the binary with `crystal build ./src/whichttp.cr`. Run with `./whichttp <STATUS CODE>`.
